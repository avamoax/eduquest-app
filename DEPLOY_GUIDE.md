# 🚀 Deploy Your EduQuest App - Get a Shareable Link!

Follow these simple steps to get a public link for your EduQuest app that you can share with anyone!

## 🌐 Option 1: Vercel (Recommended - Free & Easy)

### Step 1: Create Vercel Account
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub, Google, or email
3. It's completely free!

### Step 2: Deploy Your App
1. **Upload your project**:
   - Drag and drop your `eduquest-app` folder to Vercel
   - OR connect your GitHub repository

2. **Configure settings**:
   - Framework: **Vite**
   - Build command: `npm run build`
   - Output directory: `dist`

3. **Click Deploy** 🚀

### Step 3: Get Your Link!
After deployment (takes 1-2 minutes), you'll get a link like:
```
https://eduquest-app-xyz123.vercel.app
```

**This link works anywhere in the world! 🌍**

## 🌐 Option 2: Netlify (Alternative - Also Free)

### Step 1: Create Netlify Account
1. Go to [netlify.com](https://netlify.com)
2. Sign up for free

### Step 2: Deploy
1. Drag your `eduquest-app/dist` folder to Netlify
2. Or connect your GitHub repository
3. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`

### Step 3: Get Your Link!
You'll get a link like:
```
https://amazing-eduquest-abc123.netlify.app
```

## 🌐 Option 3: GitHub Pages (Free with GitHub)

### Step 1: Push to GitHub
1. Create a GitHub repository
2. Upload your `eduquest-app` code

### Step 2: Enable GitHub Pages
1. Go to repository Settings
2. Scroll to "Pages" section
3. Select source: GitHub Actions
4. Use the Vite deployment action

### Step 3: Get Your Link!
```
https://yourusername.github.io/eduquest-app
```

## ⚡ Quick Deploy (Easiest Method)

### Using Vercel CLI:
```bash
# Install Vercel CLI
npm i -g vercel

# In your eduquest-app folder
cd eduquest-app
vercel

# Follow the prompts
# Get instant link!
```
